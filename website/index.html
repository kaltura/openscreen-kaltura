Built on OpenScreen

This is a community fork of OpenScreen, a fantastic open-source screen recorder created by Sid and a growing community of contributors. We loved everything about it — the clean UI, the editing tools, the attention to detail. The only thing missing was cloud sharing. So we added a provider-agnostic Cloud menu with Kaltura as the included provider.

Only for cloud sharing (save and load from cloud). Without an account, the app works identically to the original OpenScreen — record, edit, and export locally. Cloud is a publishing step, not a dependency.
You can sign up directly from the app's settings dialog, or start a free trial here.
Once uploaded, your recordings are stored in your Kaltura account. They benefit from Kaltura's transcoding, adaptive streaming, analytics, and AI features. You can access them from any machine where you sign in.
Yes. Session tokens are encrypted on disk using your system's secure storage. Passwords are never written to disk — only cached in memory with a time limit. All API calls use timeouts, and file operations are path-validated to prevent traversal attacks.
A provider-agnostic Cloud menu for saving and loading recordings. Kaltura ships as the included provider. Everything else — recording, editing, exporting — is identical to upstream. The architecture is open to adding other cloud providers.
